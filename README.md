# pakistan-education-analytics-dashboard
A Power BI dashboard developed to analyze educational performance across Pakistan using district-level data from 2013–2016. The project applies Descriptive, Predictive, and Prescriptive Analytics to identify educational disparities, evaluate key influencing factors, and support data-driven decision-making.
# Features
Province and district-level education analysis
Enrollment and gender parity insights
School infrastructure assessment
Education score forecasting and trend analysis
What-If scenario modeling for policy decisions
Interactive drill-down and filtering capabilities
# Technologies Used
Power BI
Power Query
DAX
Data Modeling (Star Schema)
# Key Insights
Significant educational disparities exist across provinces and districts.
School infrastructure strongly impacts educational outcomes.
Gender inequality remains a challenge in several regions.
Security improvements correlate with better education performance.
